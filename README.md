# 고양이 토닥토닥 MVP

모바일 타겟의 간단한 정적 SPA입니다. 치즈냥이(`cheese`)와 똘순이(`ddol`)를 고른 뒤 화면 하단의 좌우 터치 영역을 누르면 `image/` 폴더의 `C`, `L`, `R` 사진으로 상태가 전환됩니다.

## 실행

Git Bash 기준:

```bash
cd /c/dev/cat-patpat
start index.html
```

또는 `index.html` 파일을 브라우저로 직접 열면 됩니다.

## 파일 구조

```text
cat-patpat/
  image/
    cheese-C.png
    cheese-L.png
    cheese-R.png
    ddol-C.png
    ddol-L.png
    ddol-R.png
  index.html
  styles.css
  app.js
  README.md
```
